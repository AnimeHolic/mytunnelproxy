# My Tunnel Proxy Website

Vercel-ready Next.js site for My Tunnel Proxy with:

- a Play Store-inspired homepage
- a dedicated privacy policy page
- locally stored Play Store icon and screenshots

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run start
```

## Deploying to Vercel

1. Import this folder as a new Vercel project.
2. Keep the default Next.js framework detection.
3. Deploy with the standard build command: `npm run build`.

No custom Vercel configuration is required.
